Move USDC to Arc

Move any amount of USDC from a supported source chain to Arc. This tool is separate from invoice checkout, so you choose the amount and source yourself.

Phase 1 — Add USDC to GatewayPrepare
Use this only when Gateway ready balance is not enough. This prepares USDC from source chains; it does not mint USDC on Arc yet.
Phase 1 can use Auto or one selected source chain. Phase 2 auto-routes from Gateway ready balances.
Phase 2 — Move Gateway USDC to ArcMint
Use this when Gateway ready balance is enough. This signs the Gateway spend/mint to Arc.
Source chain is selected automatically from Gateway ready balances to avoid choosing the wrong chain. Pay Invoice stays Arc-only.
Standalone tool
Not checked yet. Refresh balances to see whether you need Phase 1 or can continue with Phase 2. Move USDC to Arc first, then return to Pay Invoice to submit payment. Read the movement guide.
